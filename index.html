Sobre

Sou um desenvolvedor em formação, atualmente cursando Análise e Desenvolvimento de Sistemas (4º semestre). Tenho me dedicado a construir uma base sólida em desenvolvimento Front-End, utilizando HTML, CSS e JavaScript, além de aplicar versionamento com Git e GitHub em projetos práticos disponíveis no meu GitHub.

Atualmente, também estou ampliando meus conhecimentos em Java e desenvolvimento Full Stack, por meio de cursos, bootcamps e projetos que reforçam minhas habilidades em estruturação de código, responsividade e interatividade.

Tenho grande interesse em boas práticas de desenvolvimento, organização de código e criação de interfaces intuitivas e eficientes, sempre buscando escrever código limpo e de qualidade. Meu objetivo é conquistar minha primeira oportunidade como Desenvolvedor Front-End, evoluir profissionalmente e, futuramente, consolidar minha carreira como Desenvolvedor Java Full Stack.
